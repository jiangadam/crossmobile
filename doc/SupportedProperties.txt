ios.device : Device size supported
  iphone : Resolution 320x480
  ipad   : Resolution 768x1024

ios.splash.delay : Delay time of splash screen. Values in seconds. Defaults to 2.

ios.map.apikey : The API key for the MapActivity. Should match the current key used.

ios.tablecells : The number of different hashed tablecells. The bigger this number is, the more caching is performed. Defaults to 20.
